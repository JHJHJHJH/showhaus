# Showhouse API Bruno Collection

This directory contains the Bruno collection for the Showhouse API.

## How to use

1.  Install [Bruno](https://www.usebruno.com/).
2.  Open Bruno and click on "Open Collection".
3.  Select this `bruno` directory.
4.  Select the `local` environment from the environment dropdown in the top right corner.
5.  Run requests!

## Endpoints

- **App**: General app endpoints (Hello, Env, Transformed).
- **UraPrivateResi**: Endpoints for creating and retrieving ura-private-resis.
- **Scrape**: Endpoints for triggering URA API scraping.
